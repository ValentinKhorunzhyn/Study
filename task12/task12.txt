Create a class that uses static initialization blocks,
 instance initialization blocks and a constructor,
 and prove the sequence in which they get called when the class is invoked and a new object is created.


 Создайте класс который использует статические блоки инициализации, пример блока инициализации и конструктор,
 что то там последовательность в которой они вызываются когда класс вызван и новый объект создан.
